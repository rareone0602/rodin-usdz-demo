Rodin USDZ Demo

Use your iPhone/iPad and Safari to get full experience. Tap on the icon at the upper-right corner to view in AR.

This 3D asset is AI-generated, forked from Ernie Chu's project.

Rodin is a 3D asset generator that takes as input images and text prompts to generate corresponding 3D assets. It supports USDZ export, meaning that the asset can be presented in a webpage via Apple's built-in API. (You may need to re-export the asset with Reality Converter or other alternatives due to format discrepancy.) Checkout the GitHub repo for more information!
